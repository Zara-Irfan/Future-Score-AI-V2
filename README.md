# FutureScore AI (Version 2 - Node.js)

AI-powered career assistant for ATS resume optimization and personalized career guidance.

## Features
- ATS resume scoring against job descriptions
- AI analysis with detailed breakdowns
- Personalized career roadmap with free courses
- 2026-ready insights on emerging technologies

## Setup (Node.js)
1. Clone the repo
2. Install dependencies: `npm install`
3. Set environment variable: `export GROQ_API_KEY=your_key_here`
4. Run: `npm start`

## Deployment
Deployed on Vercel.

## Tech Stack (Version 2)
- Node.js
- Express
- Groq AI
